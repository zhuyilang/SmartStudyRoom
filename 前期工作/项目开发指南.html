📚 智能校园自习室预约管理平台

项目开发指南 · 团队协作手册

一、项目概况

做一个校园自习室预约网站，解决占座、空位不明的痛点。 学生可以在线查看自习室座位、预约选座、签到使用； 管理员可以管理校区/楼栋/自习室/座位，查看使用率报表和热力图。

技术栈总览

Spring Boot 2.7.18 MyBatis-Plus 3.5.3 MySQL 8.0 Maven Knife4j Lombok Vue 3 Vite Element Plus Axios Vue Router Pinia ECharts

🔧 后端工具

  • IntelliJ IDEA (Community)
  • Navicat / DBeaver
  • Postman

🎨 前端工具

  • VS Code
  • Node.js 18+
  • Chrome 开发者工具

三、数据库 8 张表

表名作用说明
user用户表学生 + 管理员，role 字段区分
campus校区表如：主校区、东校区
building楼栋表外键关联 campus
floor楼层表外键关联 building
room自习室表外键关联 floor，含行数列数
seat座位表外键关联 room，status: 0空闲/1占用/2维修
reservation预约表外键关联 user+seat+room，status: 0已预约/1已签到/2已取消/3已超时/4已完成
blacklist黑名单表外键关联 user，纪录爽约次数和封禁时间
💡 不需要手动建库建表！ 启动后端时会自动创建数据库和所有表，并插入测试数据。
